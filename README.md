# Pets Spa

Адаптивная верстка сайта для салона красоты для животных (Pet Spa).

## О проекте

Данный проект представляет собой лендинг салона для ухода за животными.
Сайт был сверстан по обучающему видео в рамках практики по frontend-разработке.

## Используемые технологии
HTML5
CSS3 (Flexbox, Grid)
Современный CSS (вложенные селекторы)
Swiper Slider (слайдер)

## Реализовано
Адаптивная верстка (под десктоп, планшеты и смартфоны)
Мобильная навигация
Секция услуг
Секция с изображениями
Слайдер (Swiper)
Чистая и структурированная разметка

## Как запустить
1. Скачать или клонировать репозиторий
2. Открыть файл index.html в браузере
